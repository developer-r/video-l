# Movie Parser Plugin для Lampa

Плагин для поиска и просмотра фильмов/сериалов на Smart TV через приложение Lampa.

## Возможности

- 🔍 **Поиск фильмов** через TMDB API
- 📺 **Просмотр трейлеров** с YouTube
- 🎬 **Интеграция** с различными видеоисточниками
- 🖥️ **Поддержка** Smart TV (Samsung, LG, Android TV)

## Установка

### Через URL (основной способ)

Lampa загружает плагины напрямую из интернета. Вам нужно:

1. **Загрузить файл плагина** на хостинг (GitHub Gist, GitHub Raw, или любой веб-сервер)
2. **Получить прямую ссылку** на файл `.js`
3. **Добавить в Lampa**:
   - Откройте Настройки → Плагины
   - Выберите "Добавить плагин по ссылке"
   - Вставьте URL: `https://raw.githubusercontent.com/ВАШ_НИК/ВАШ_РЕПОЗИТОРИЙ/main/movie_parser.js`

### Пример с GitHub Gist

1. Создайте Gist на https://gist.github.com с содержимым `movie_parser.js`
2. Получите "Raw" ссылку на файл
3. Используйте эту ссылку в Lampa

### Локальная установка (для разработки)

1. Создайте папку `plugins` в корне приложения Lampa
2. Скопируйте файлы плагина
3. Перезапустите Lampa

## Использование

### Поиск фильмов

1. Откройте приложение Lampa
2. Перейдите в поиск (кнопка поиска на пульте)
3. Введите название фильма
4. Выберите источник "TMDB" из списка
5. Выберите фильм из результатов

### Просмотр

1. Нажмите на найденный фильм
2. Выберите "Смотреть" из списка источников
3. Выберите качество/источник видео
4. Начнётся воспроизведение

## Настройка TMDB API

Плагин использует TMDB API для получения информации о фильмах. По умолчанию используется демо-ключ.

Для использования собственного API ключа:

1. Зарегистрируйтесь на https://www.themoviedb.org/
2. Получите API ключ в настройках профиля
3. Отредактируйте файл `movie_parser.js`, замените значение `tmdb_api_key`:

```javascript
tmdb_api_key: 'ВАШ_API_КЛЮЧ',
```

## Добавление новых источников видео

### Через video_sources.js

Добавьте новый источник в объект `sources`:

```javascript

        name: 'Название источника',
        embed: 'https://example.com/embed/',
        type: 'iframe' // или 'api'

```

## Ограничения

⚠️ **Важные ограничения:**

1. **CORS** - Браузеры и TV приложения имеют ограничения на прямые запросы к сторонним сайтам
2. **Для полноценной работы** рекомендуется использовать прокси-сервер
3. **Бесплатные источники** могут работать нестабильно

### Решение проблемы CORS

Для обхода ограничений CORS используйте:

1. **Прокси-сервер** - Сервер, который делает запросы к сайтам и возвращает результат
2. **Расширение браузера** - Для тестирования на ПК
3. **CORS-прокси** - Публичные прокси (не рекомендуется для продакшена)

Пример простого прокси на Node.js:

```javascript
const express = require('express');
const request = require('request');
const app = express();

app.get('/proxy', (req, res) => {
    const url = req.query.url;
    request(url).pipe(res);
});

app.listen(3000);
```
