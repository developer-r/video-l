# MovieParser TMDB для Lampa

Переписанная с нуля версия плагина. Основной файл для подключения только один: `movie_parser.js`.

## Что делает плагин

- добавляет источник поиска `TMDB` в поиск Lampa;
- добавляет кнопку `TMDB` на карточке фильма;
- открывает отдельный экран с результатами поиска из TMDB.

## Какие файлы использовать

- `movie_parser.js` — основной и единственный обязательный файл;
- `video_sources.js` — безопасный helper, подключать не нужно.

## Как добавить плагин в Lampa

### Вариант 1. Через GitHub

1. Залей файл `movie_parser.js` в репозиторий или Gist.
2. Возьми именно прямую `Raw`-ссылку на файл.
3. В Lampa открой:
   `Настройки -> Расширения / Плагины -> Добавить плагин`
4. Вставь `Raw`-ссылку и подтверди установку.

Пример правильного формата ссылки:

```text
https://raw.githubusercontent.com/USERNAME/REPOSITORY/main/plugins/movie_parser.js
```

### Вариант 2. Локально

1. Скопируй `movie_parser.js` на доступный тебе веб-сервер.
2. Получи прямую ссылку на сам `.js` файл.
3. Добавь эту ссылку в Lampa через меню плагинов.

## Важно

- Не указывай ссылку на HTML-страницу GitHub или Gist, нужна именно `Raw`-ссылка.
- Не подключай папку целиком, Lampa ожидает URL на конкретный `.js` файл.
- Для этой версии достаточно подключить только `movie_parser.js`.

## Если снова будет ошибка

Открой в Lampa:

`Меню -> Консоль`

И посмотри сообщения с префиксом:

```text
[MovieParser]
```

## Почему предыдущая версия ломалась

- один из файлов мог обращаться к `Lampa` слишком рано;
- инициализация зависела от порядка загрузки;
- было несколько потенциальных точек падения при старте.

В новой версии запуск сделан безопаснее: плагин ждет готовности Lampa и не требует второго файла для основной работы.
